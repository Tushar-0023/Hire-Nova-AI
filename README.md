# HireNova AI 🚀

HireNova AI is a full-stack recruitment platform that connects recruiters and candidates through a modern web interface. It allows recruiters to post and manage jobs, while users can explore opportunities and apply بسهولة.

## 🎯 Features

### 👤 User (Candidate)
- Register & Login
- Browse available jobs
- Apply for jobs
- View application status

### 🧑‍💼 Recruiter
- Secure login
- Post new jobs
- View applicants
- Manage job listings

### 📊 Dashboard
- Organized job and application data
- Clean and responsive UI

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Bootstrap / CSS

### Backend
- Node.js
- Express.js

### Database
- MySQL

---

## ⚙️ How It Works

- Users and recruiters log in through role-based authentication  
- Recruiters create job postings  
- Users apply for jobs  
- Applications are stored and managed through the backend  
- Dashboard displays structured data for easy access
